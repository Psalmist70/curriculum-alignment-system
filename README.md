Functional Requirements (what the system must do)

Store curriculum data
Load job market data (from dataset or scraping)
Run analysis (NLP processing)
Compare curriculum vs job skills
Identify skill gaps
Display results on dashboard

Fast processing for small datasets
Easy-to-use interface
Accurate text analysis
Reusable (can run analysis multiple times)
